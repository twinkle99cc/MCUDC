桃銘熱舞總共有 5 種舞風：H、J、L、P、B。

不論是否有基礎，只要你熱愛街舞，這裡都有屬於你的舞台。

Locking 組

Locking 是一種源自 1970 年代美國的街舞風格，屬於 Funk Style 的一種。

它的特色是動作突然鎖住（Lock），結合大幅度的手部與肢體動作，常帶有喜劇感與互動性。

Breaking 組

Breaking 是最早的街舞風格之一，融合了地板動作、空翻技巧和強烈的節奏感。

不僅展現力量與技巧，更是展現個人風格與創造力的最佳舞台。
